# Project SCTM: Research Edition

This repository contains the source code for the SCTM (Sparse Tensor Memory) research presentation website.

## Structure

- `index.html`: Main entry point.
- `css/styles.css`: Custom styles and animations.
- `js/script.js`: Logic for simulation, math engine, and charts.

## Deployment

This static site is ready for deployment on Vercel, Netlify, or GitHub Pages.

### Local Development

Simply open `index.html` in your browser.
